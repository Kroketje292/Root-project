Hoofdvraag

Belangrijk in een project is om een rode draad te hebben. De hoofdvraag is de centrale onderzoeksvraag die je wilt beantwoorden in een onderzoek of een project. Het is de vraag waar alles om draait en die je probeert te beantwoorden door middel van onderzoek



De hoofdvraag:

“Hoe kan Root de huidige verouderde SharePoint-server vervangen door een nieuwe documentatieoplossing die zowel administratieve als technische documentatie kan bijhouden”
